1) Коэффициент a равно 0 (a=0, b!=0, c!=0) => вывод одного корня.
2) Коэффициенты a и b равны 0 (a=0, b=0, c!=0) => сообщение "Нет корней".
3) Все коэффициенты равны 0 (a=0, b=0, c=0) => сообщение "R".
4) Коэфициенты а не равен 0
4.1) Дискриминант меньше 0 (d<0) => сообщение "Нет корней"
4.2) Дискриминант равен 0 (d=0) => вывод одного корня.
4.3) Дискриминант больше 0 (d>0) => вывод двух корней.
